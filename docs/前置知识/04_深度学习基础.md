# 深度学习基础

## 需要掌握的知识点

- PyTorch 基础：Tensor、Module、优化器
- MLP 网络：全连接网络结构与前向传播

## 为什么对人形行走训练重要

- 策略网络与价值网络通常由 MLP 构成
- 模型结构和训练稳定性会影响收敛速度与步态质量
- 导出部署（TorchScript/ONNX）需要基本框架知识

## 学习建议

- 先会写一个最小 MLP 分类/回归例子
- 再理解 Actor-Critic 的双网络结构
- 能读懂训练脚本里的 optimizer、learning rate、batch 配置
