# 机器人学基础

## 需要掌握的知识点

- 正/逆运动学：关节角度与末端位置之间的映射关系
- 刚体动力学：牛顿-欧拉方程、拉格朗日方程
- 雅可比矩阵：关节速度到末端速度的线性映射

## 为什么对人形行走训练重要

- 运动学帮助你理解“动作输出到关节后，机体会怎么动”
- 动力学帮助你理解“为什么会摔倒/抖动/耗能高”
- 雅可比帮助你理解“身体姿态变化和关节协同关系”

## 学习建议

- 先掌握二维平面机械臂的正逆运动学，再迁移到腿部关节链
- 对髋-膝-踝三关节先做简化模型推导
- 结合 Isaac Sim 可视化观察关节变化与机体姿态变化
