# 前置知识总览

本文档整理自 `04_自定义机器人训练指南.md` 的 `6.1~6.4`，用于在正式训练前做知识补齐。

## 分类目录

- `01_机器人学基础.md`
- `02_强化学习基础.md`
- `03_控制理论基础.md`
- `04_深度学习基础.md`

## 建议学习顺序

1. 机器人学基础（先建立“机器人怎么动”的直觉）
2. 控制理论基础（理解“如何稳”）
3. 强化学习基础（理解“如何学会走”）
4. 深度学习基础（理解“策略网络如何实现”）

## 目标

- 能读懂 Isaac Lab 中 locomotion 任务的观测、动作、奖励设计
- 能理解 PPO 训练日志中的核心指标
- 能对步态不稳问题做基础定位（控制参数/奖励项/网络侧）
