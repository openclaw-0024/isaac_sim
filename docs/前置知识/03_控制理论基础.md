# 控制理论基础

## 需要掌握的知识点

- PD 控制：位置控制的基础
- 阻抗控制：力-位置混合控制
- 全身控制（WBC）：任务空间目标到关节空间控制量映射

## 为什么对人形行走训练重要

- RL 策略通常输出目标关节量，底层仍依赖控制器执行
- 控制参数与步态稳定性、抖动、能耗高度相关
- 实机部署时，控制器质量直接影响 sim2real 成败

## 学习建议

- 先从单关节 PD 响应曲线（超调、稳态误差）入手
- 再到双足站立平衡，观察不同增益的稳定域
- 在 Isaac Lab 中对比不同 stiffness/damping 的行为差异
