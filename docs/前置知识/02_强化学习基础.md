# 强化学习基础

## 需要掌握的知识点

- MDP：状态、动作、转移概率、奖励
- 策略梯度：REINFORCE、Actor-Critic
- PPO：clip 机制、优势估计
- Reward Shaping：奖励函数设计方法

## 为什么对人形行走训练重要

- 行走任务本质是序列决策，MDP 是统一建模语言
- PPO 是 Isaac Lab 中常用 locomotion 算法
- 奖励函数会直接决定机器人学到的步态风格

## 学习建议

- 先用 CartPole 建立 PPO 训练直觉，再看人形任务
- 看训练曲线时重点关注 reward、episode length、loss 变化
- 调参时每次只改一个变量（例如只改 `track_lin_vel` 权重）
