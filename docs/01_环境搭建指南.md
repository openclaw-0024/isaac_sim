# 01 - 环境搭建指南

> 预计耗时：1 周 | 前置条件：满足硬件最低配置

---

## 1. 系统要求

| 项目 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 操作系统 | Ubuntu 22.04 / Windows 11 | Ubuntu 22.04 |
| GPU | RTX 3080 (16GB VRAM) | RTX 4090 (24GB VRAM) |
| CPU | i7-12700 / Ryzen 9 5900X | i9-13900K / Ryzen 9 7950X |
| 内存 | 32GB | 64GB DDR5 |
| 存储 | 500GB SSD | 1TB NVMe SSD |
| NVIDIA 驱动 | 580.65.06+ (Linux) / 580.88+ (Win) | 最新生产分支 |
| CUDA | 12.x | 跟随 Isaac Sim 版本 |

> **重要**：GPU VRAM 低于 16GB 无法运行 Isaac Sim。GTX 系列不支持。

---

## 2. 安装步骤（Linux / Ubuntu 22.04 推荐）

### 2.1 安装 NVIDIA 驱动

```bash
# 检查当前驱动
nvidia-smi

# 如需更新驱动（Ubuntu）
sudo apt update
sudo apt install nvidia-driver-580
sudo reboot
```

### 2.2 安装 Miniconda

```bash
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
# 重启终端后验证
conda --version
```

### 2.3 创建 Conda 环境

```bash
conda create -n isaaclab python=3.11 -y
conda activate isaaclab
```

> Isaac Sim 5.x 要求 Python 3.11；Isaac Sim 4.x 要求 Python 3.10。

### 2.4 安装 Isaac Sim

```bash
pip install isaacsim==5.1.0
```

### 2.5 克隆并安装 Isaac Lab

```bash
git clone https://github.com/isaac-sim/IsaacLab.git
cd IsaacLab

# Linux
./isaaclab.sh --install

# Windows
# isaaclab.bat --install
```

### 2.6 验证安装

```bash
# 运行基础测试
./isaaclab.sh -p source/isaaclab/test/test_environment.py

# 尝试启动一个简单场景（需要 GUI）
./isaaclab.sh -p source/isaaclab/demos/demo_scene.py
```

## 3. 安装步骤（Windows 11）

### 3.1 前置软件

- 安装 Visual Studio Build Tools 2022（勾选 "C++ 桌面开发"）
- 安装 Git for Windows
- 安装 Miniconda（Python 3.11）

### 3.2 安装流程

```powershell
# 创建环境
conda create -n isaaclab python=3.11 -y
conda activate isaaclab

# 安装 Isaac Sim
pip install isaacsim==5.1.0

# 克隆 Isaac Lab
git clone https://github.com/isaac-sim/IsaacLab.git
cd IsaacLab

# 安装
isaaclab.bat --install
```

## 4. 云端替代方案

如果本地 GPU 不满足要求，可以使用以下云平台：

### 4.1 NVIDIA Brev（官方推荐）

- 地址：https://www.brev.dev/
- 预装 Isaac Sim + Isaac Lab，开箱即用
- 按需付费，适合学习阶段

### 4.2 AWS / GCP

- 推荐实例类型：AWS g5.2xlarge (A10G 24GB) 或 p4d (A100)
- 需要自行配置环境
- 成本较高，适合大规模训练

### 4.3 AutoDL / 恒源云（国内）

- 搜索带有 RTX 4090 / A100 的 GPU 实例
- 需要自行安装 Isaac Sim

## 5. 常见问题

| 问题 | 解决方案 |
|------|---------|
| `libGL error` | `sudo apt install libgl1-mesa-glx` |
| Isaac Sim 启动黑屏 | 检查 GPU 驱动版本，确保 >= 580 |
| 内存不足 OOM | 减少 `--num_envs` 数量 |
| Windows 路径过长 | 将 IsaacLab 克隆到短路径如 `C:\IsaacLab` |
| conda 环境冲突 | 使用 `conda create` 全新环境，不要复用 |
